# qq_music_spa

> A Vue.js project

## Build Setup

``` bash
# 安装依赖
npm install

# serve with hot reload at localhost:8080 启动服务
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
